All Release Notes

Version 1.8 (26. March 2013).

  • Issue 74: Symmetric encryption
    One of the most requested GPGServices features has arrived. You can now encrypt with a password.
  • Automatically informs you of new versions
    The same update process you're familiar with from GPGMail is now included in GPGServices to always keep you up to date.
  • Issue 73: Encrypt read-only text selections
    Now GPGServices will work in most used browsers like Safari, Chrome, Firefox, Chromium and more as well as in Mail.app.
  • Issue 116: Added Version information
    Now displaying the version of GPGServices being used so users easily can find out which version they have.
  • Issue 146: Fix for double encryption / decryption
  • Issue 131: Fix for signature not being found
  • Issue 121: Fix broken Growl icon
  • Issue 125: Fix for canceling encryption resulting in finished encryption
  • Issue 102: Display missing error messages to the user
  • Issue 155: Fix for PathFinder creating duplicate files when decrypting and some other minor oddities with PathFinder
  • Load the progress window in languages other than english.
  • Checksum: 5836bf9585b89db2f65f08673a7110453f32e5d0 (SHA-1)
